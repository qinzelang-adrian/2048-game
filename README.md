# 2048 Game

一个使用 Python + pygame 实现的 2048 小游戏，带有平滑的移动/合并动画和悔棋功能。

## 功能特点

- 🎮 方向键控制方块移动与合并
- ✨ 平滑的滑动、合并、生成动画，合并时还有光晕特效
- ⏪ 悔棋功能，最多可回退 50 步
- 🏆 游戏结束 / 胜利（凑出 2048）自动判定
- 🔄 一键重新开始

## 操作说明

| 按键 | 功能 |
| --- | --- |
| ↑ ↓ ← → | 移动方块 |
| Z | 悔一步 |
| R | 重新开始 |
| 空格 | 游戏结束/胜利后开始新一局 |

## 运行方式

### 方式一：运行源代码

需要先安装 Python 3 和 pygame：

```bash
pip install pygame
python 2048游戏.py
```

### 方式二：直接运行

在 [Releases](../../releases) 页面下载打包好的 `2048游戏.exe`（仅支持 Windows），双击即可运行，无需安装 Python 环境。
